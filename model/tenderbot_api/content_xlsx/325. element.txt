Войти на Портал поставщиков можно несколькими способами: 
Вход по логину и паролю пользователя
1. Нажать на кнопку "Войти", расположенную в верхней части главной страницы Портала поставщиков: 

2. В поле "Логин" указать электронную почту, указанную у пользователя в профиле, а также пароль пользователя: 

3. Нажать кнопку "Войти":

4. После чего откроется главная страница Портала поставщиков, а в шапке Портала будут отображены данные пользователя: 

 
 
Вход по электронной подписи 
1. Нажать на кнопку "Войти", расположенную в верхней части главной страницы Портала поставщиков: 

2. Нажать кнопку "ЭП": 

3. Выбрать сертификат пользователя, под которым необходимо войти: 

4. Нажать кнопку "Войти":

5. После чего откроется главная страница Портала поставщиков, а в шапке Портала будут отображены данные пользователя: 

 
 
Вход с помощью внешней системы
1. Нажать на кнопку "Войти", расположенную в верхней части главной страницы Портала поставщиков: 
 

2. Выбрать систему, с помощью которой необходимо осуществить вход на Портал поставщиков: 
 
3. После чего откроется страница авторизации во внешней системе, где необходимо ввести учетные данные и авторизоваться.
Пример окна авторизации в внешней системе "Mos.ru":

4. В случае успешной авторизации, откроется главная страница Портала поставщиков, а в шапке Портала будут отображены данные пользователя: 

 
Продолжение работы после входа с помощью внешней системы
При необходимости:

Заполнить недостающие данные о пользователе и компании
Добавить сертификат ЭП в личный кабинет пользователя
Отправить заявку на регистрацию / изменение данных о компании
