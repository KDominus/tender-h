Для смены электронной почты необходимо перейти в «Профиль пользователя» и нажать на
кнопку «Сменить E-mail».
 
На Ваш электронный адрес будет выслана инструкция по изменению почты. При смене пользователем электронной почты, будет изменен и логин пользователя, необходимый для входа на Портал поставщиков.
 
В случае,если пользователь не может самостоятельно изменить электронную почту, то необходимо обратиться в Службу технической поддержки Портала поставщиков.
