У каждой организации с разными реквизитами будет свой Личный кабинет. Система пропустит регистрацию двух организаций с одним кодом ИНН и разными кодами КПП.
Для регистрации филиала требуется наличие электронной подписи с данными филиала (ИНН и КПП). Далее необходимо пройти регистрацию нового пользователя через главную страницу Портала поставщиков. После регистрации требуется подать заявку на регистрацию компании через профиль нового пользователя.
