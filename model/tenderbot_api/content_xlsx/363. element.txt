Определение победителя в закупке по потребности
В соответствии с правилами проведения закупок по потребности, заказчик самостоятельно выбирает поставщика из списка предложений согласно п. 5.1 Приложения 5 Регламента ведения Портала поставщиков.
По истечении срока подачи заявок. Заказчик из числа участников закупки осуществляет выбор предложения (оферты), условия поставки товара/оказания услуги/выполнения работы которого наиболее соответствуют потребностям заказчика
Дополнительные требования к определению победителя могут быть установлены Уполномоченным органом заказчика.
