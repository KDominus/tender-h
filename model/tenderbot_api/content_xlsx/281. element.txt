Важно! После отправки заявки на изменение данных и до утверждения заявки пользователю будут недоступны действия с котировочными сессиями, контрактами и офертами.
 
 
Для изменения сведений о компании необходимо сформировать заявку, нажав кнопку "Заявка на изменение данных" .

 
1. Изменение сведений о компании
Функционал изменения сведений компании доступен для пользователей со следующими ролями:

Администратор компании, 
Поставщик,
Уполномоченный специалист поставщика,
Специалист по работе с офертами на основе СТЕ.

При нажатии кнопки "Сохранить", данные о компании сохранятся в режиме черновика. Но при выполнении операций на Портале поставщиков будут использоваться предыдущие сведения компании. 

 
В открывшейся форме необходимо внести/скорректировать требуемые сведения и оправить заявку, нажав кнопку "Отправить заявку".
 

После подтверждения заявки модераторами измененные сведения будут изменены.
 
Данные по заявке можно посмотреть на вкладке «Отправленные заявки», для этого из раздела «Профиль компании» необходимо перейти на страницу «Заявка на изменение данных» и выбрать вкладку «Отправленные заявки»
 

 
 
Пока заявка не будет одобрена, у пользователей компании в шапке Портала рядом с изображением профиля будет отображен признак «Права ограничены», при наведении на который покажется причина ограничений. 

 
2. Загрузка данных из сертификата ЭП
Чтобы загрузить в профиль данных из сертификата ЭП, необходимо: 
 

В "Профиле компании" нажать кнопку "Заявка на изменение данных",

 
2. На странице "Заявка на изменение данных" нажать кнопку "Загрузить данные из сертификата",

3. После того, как данные из сертификата автоматически будут загружены, необходимо нажать кнопку "Отправить заявку",
 

3. Просмотр отправленных заявок
Чтобы посмотреть данные об отправленных заявках нужно:

Нажать кнопку  "Заявка на изменение данных" на странице "Профиль компании",


2. Перейти на вкладку "Отправленные заявки" на странице "Заявка на изменение данных",
 

 
На вкладке "Отправленных заявки" можно просмотреть список отправленных ранее заявок, а так же:

ФИО пользователя, который отправил запрос на изменение данных, 
Дату отправки запроса, 
Тип отправленной заявки,
Статус отправки и принятия изменений сведений о компании,
Историю изменений заявки.
