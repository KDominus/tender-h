Для выбора оферты необходимо перейти в раздел «Товары», после чего в данном каталоге указать соответствующую продукцию или услугу, после выбора откроется карточка СТЕ (стандартной торговой единицы) продукции/услуги с возможностью добавления оферты в корзину.
 


 
 
Рекомендуем выбрать наиболее подходящую, не отличающуюся по основным характеристикам оферту, либо создать котировочную сессию через страницу "Мои котировочные сессии", указав необходимые характеристики.
