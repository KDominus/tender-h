Для подписания оферт, загруженных с помощью массового импорта YML- файла, пользователю необходимо выделить оферты для подписания.
При нажатии на кнопку «Подписать оферты» осуществится переход в раздел «Управление офертами». Для массового подписания оферт необходимо нажать «Перейти к подписанию» и подписать оферты.
Оферты перейдут в статус «Зарегистрирована».
