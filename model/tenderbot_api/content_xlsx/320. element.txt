Для того, чтобы найти определенный контракт в реестре, необходимо воспользоваться фильтрами. Поля для заполнения критериев поиска скрыты в отдельной области, раскрыть которую можно с помощью кнопки под вкладками «Показать фильтры».
 

Фильтрация контрактов в реестре
Фильтры разделены на основные и расширенные.
Основные фильтры
 

С помощью основных фильтров можно найти контракты по следующим критериям:



Наименование поля поиска
Описание поля поиска


Регион
Поиск осуществляется по региону заказчика, с которым заключен контракт


Категория продукции
Для того, чтобы выбрать категорию, необходимо открыть модальное окно с деревом категорий (справочник КПГЗ). В дереве можно выбрать несколько значений


Сумма контракта
С помощью данного фильтра можно найти контракты, сумма которых входит в определенный диапазон. Для этого необходимо ввести значения в поле «От» и/или «По»


Статус
В публичном реестре можно просматривать и фильтровать контракты, находящиеся в статусах «Заключен», «Исполнен» и «Расторгнут». Посмотреть собственные контракты можно в разделе "Мои контракты"



Расширенные фильтры
 

С помощью основных фильтров можно найти контракты по следующим критериям:



Наименование поля поиска
Описание поля поиска


Предмет контракта
Указывается наименование (или часть) закупаемой продукции


Дата заключения контракта
Указывается период поиска по дате заключения контракта


Основания для заключения контракта
Указывается одно или несколько оснований заключения контракта 


Способ размещения закупки 
Указывается один или несколько способов размещения закупки, по результатам которой был заключен контракт


Дата действия контракта
Указывается диапазон поиска по дате действия контракта


Номер оферты
Указывается номер оферты, по которой ранее были заключены контракты


СТЕ
В данном поле можно осуществить поиск СТЕ, по которой ранее были заключены контракты. Поиск происходит по номеру или наименованию СТЕ


ОКПД2
Указывается одна или несколько позиций справочника ОКПД2


Номер контракта 
Указывается номер контракта, который необходимо найти


Реестровый номер контракта
Указывается реестровый номер контракта, который необходимо найти


Размещение сведений 
Выбирается один или несколько законов основания заключения контрактов


Организационные признаки

Результаты поиска будут включать в себя поставщиков, обладающих хотя бы одним из выбранных признаков:

СМП (Субъекты малого и среднего предпринимательства),
УИС (Предприятие, относящееся к уголовно-исполнительной системе),
Организация инвалидов,
Социально-ориентированные организации.




Номер закупки
Указывается номер закупки, по результатам который был заключен контракт 


Поставщик
Указывается поставщик(и), чьи контракты необходимо найти


Тип заключения 
Указывается формат заключения контракта - в бумажном или электронном виде (на Портале поставщиков) 


Заказчик
Указывается заказчик(и), чьи контракты необходимо найти



Карточки контрактов в реестре
Сортировка карточек в реестре
Чтобы отсортировать карточки контрактов, необходимо в верхней части страницы нажать на поле "Сортировать" и затем выбрать один из режимов сортировки:



Режим сортировки
Описание


По релевантности
Сортировка в данном режиме осуществляется по статусам (сначала показываются активные закупки, затем неактивные), внутри данного режима происходит сортировка по дате публикации. 


По сумме контрактов
Сортировка в данном режиме осуществляется по сумме контракта


По дате заключения контрактов
Сортировка в данном режиме осуществляется по дате и времени, когда контракт был заключен



 
Сортировка может осуществляться как по возрастанию, так и по убыванию.

Тип контракта

В реестре отображаются следующие типы контрактов:

контракты по итогам котировочных сессий,
контракты по итогам закупок по потребностям,
контракты по итогам прямых закупок,
контракты по итогам конкурентных процедур.

Реестровый номер контракта

Предмет контракта

Статус контракта
Статус контракта 


Сумма контракта

Наименование заказчика
Наименование заказчика, заключившего данный контракт 


Наименование поставщика
Наименование поставщика, заключившего данный контракт


Регион

Закон

Дата заключения
