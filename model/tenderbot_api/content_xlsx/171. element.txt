1) Если деньги по контракту получены в полном объеме, т.е. Заказчик заплатил по документам, достаточным для него для предоставления оплаты, то Поставщик свою работу по контракту закончил.
2) Если Поставщик выполнил контракт, предоставил документы, достаточные для получения оплаты, Заказчик принял документы, но не предоставил оплату, а контракт в статусе "Исполняется", то пишите в поддержку Портала поставщиков - либо проработаем техническую проблему, либо эскалируем дисциплинарную проблему 3) Если в закупке указано "Обязательное электронное исполнение с использованием УПД" (т.е. контракт должен быть исполнен и закрыть при помощи ЭДО), но Заказчик сопротивляется ЭДО / работает только с бумажными Актами / закрыл контракт без применения УПД, а Поставщик выполнил обязательства и получил соответствующую оплату, то за нарушение требования закупки о применении ЭДО несет ответственность Заказчик
