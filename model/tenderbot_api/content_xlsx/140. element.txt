После регистрации нового пользователя, ему необходимо присвоить полномочия для работы на Портале поставщиков.
 
Присвоить права новому пользователю может пользователь с полномочиями Администратора компании.
 
Для предоставления полномочий необходимо войти в профиль Администратора компании, далее перейти в раздел «Профиль компании» => «Сведения о пользователях».
 
В данном разделе необходимо нажать «Изменить» и добавить права из выпадающего списка.
