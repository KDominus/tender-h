Полученный черновик шаблона в формате "*docx" будет содержать в себе "элементы управления содержимым" (служебные метки). Теперь документ необходимо открыть в текстовом редакторе MS Word.
Для доступа к «элементам управления» необходимо перейти на вкладку «Разработчик» (1) и нажать на кнопку «Область сопоставления XML» (2)  
 

Если по какой-то причине в вашем MS Word отсутствует вкладка «Разработчик», то её необходимо подключить. Процесс добавления вкладки описан в разделе «Настройка MS Word перед работой» текущего руководства.
 
После нажатия на кнопку появится новая панель "Сопоставление XML". На ней в выпадающем списке необходимо выбрать пункт «Данные_Портала_Поставщиков», который был добавлен системой подготовки документов на предыдущем шаге.
 

Если указанный пункт отсутствует, то необходимо повторить все шаги начиная с раздела "Загрузка документа".
На панели отобразятся все доступные для работы "Элементы управления". 
 

Элементы управления доступные в вашем шаблоне документа могут отличаться от представленных выше, а их наличие, количество и структура зависят только от выбранного "типа документа", используемого при создании шаблона.
"Элементы управления содержимым" условно можно разделить на два типа: "Обычные" и "Повторяющиеся". Обычные элементы управления могут быть вставлены в любую часть шаблона, и при создании документа вместо них будет автоматически подставляться информация из соответствующего им типа документа.
 

Для вставки в шаблон «обычного элемента управления» необходимо нажать правой кнопкой мыши по выбранному элементу и перейти в контекстном меню: "Вставить элемент управления содержимым" - "Обычный текст".
 

 
 
В место куда был установлен курсор добавиться выбранный элемент управления.
 

Если в выбранный элемент управления не установлено тестовое значение на уровне типа документа, который использует ваш шаблон, то вставленная служебная метка будет отображать сообщение: "Место для ввода текста".
