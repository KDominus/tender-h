Прямая закупка - способ закупки, при котором контракт (договор) заключается с конкретным поставщиком без проведения дополнительных процедур. Для создания закупки данным способом необходимо заказчику выбрать оферты путем нажатия на кнопку «в корзину». Обращаем внимание, что по решению УО, РУО может быть предусмотрена обязанность  формирования предмета закупки только путем одновременного включения товаров, работ, услуг относящихся к одной категории Классификатора предметов государственного заказа. Подробнее о создании закупки данным способом, Вы можете ознакомиться в статье «Как  заказчику создать прямую закупку»
