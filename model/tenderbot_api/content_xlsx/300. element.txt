В разделе "Мои контракт" реализована возможность по регистрации на Портале поставщиков контрактов (договоров), которые были заключены напрямую вне Портала поставщиков.
При нажатии на кнопку "Зарегистрировать контракт" в реестре "Мои контракты" осуществляется переход на одноименную страницу.
 

Страница содержит следующие блоки для заполнения:
 



Наименование блока
Описание блока


Основная информация по контракту
Основная информация по контракту на бумажном носителе


Сведения о поставщике
Заполнение сведений о поставщике с возможностью выбора значений из справочника


Сведения о спецификации
Заполнение сведений о спецификации контракта на бумажном носителе


Сведения о поставке
Заполнение сведений о поставке контракта на бумажном носителе


Спецификация этапа
Заполнение сведений о спецификации этапа 


Документы контракта
Прикрепление документов контракта



 
Интерфейс страницы:

 
После заполнения всех блоков на странице регистрации контракта необходимо нажать кнопку "Зарегистрировать" .
