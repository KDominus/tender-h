Просмотр проекта контракта доступен по кнопке «Просмотр документов» в сведениях о контракте.
 
 
 По нажатию на кнопку «Просмотр документов» открывается модальное окно просмотра документов контракта.

 При просмотре документов контракта в статусе «Заключен» отображаются сведения об электронных подписях, а также доступны функциональные кнопки «Скачать документ» и «Скачать документ с подписью».
