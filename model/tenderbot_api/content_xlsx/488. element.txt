Если необходимо массово подписать оферты, созданные по котировочной сессии (далее КС), требуется перейти к списку оферт по результату КС, заполнить обязательные поля, нажав на каждой кнопку Редактировать. После заполнения обязательных полей во всех офертах нажать кнопку Подписать, произойдет подписание всех оферт (Рисунок 1)
