Срок действия предложения это дата окончания готовности участвовать в закупке по потребности.
 
Данное поле является необязательным для заполнения.
Срок действия не может быть раньше даты заключения контракта.
 
При смене поставщика (если первый участник не смог осуществить свои обязанности), будет проверяться, что указал (второй) поставщик в этом поле - если дата действия прошла, то на этого поставщика нельзя будет возложить обязанности по данной потребности.
