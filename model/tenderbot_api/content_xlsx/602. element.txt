Оферта по итогам КС формируется автоматически путем включения в неё существенных условий, отраженных в котировочной сессии и предложения победителя о цене контракта.
Поставщику доступно редактирование только полей "Артикул", "НДС" и есть возможность добавить изображение. Редактирование других полей в оферте, созданной по итогам КС, функционалом не предусмотрено.
