Для формирования структурированного документа пользователю необходимо пройти процедуру регистрации у Оператора ЭДО при исполнении первого контракта, который должен быть исполнен в рамках ЭДО. Регистрация в ЭДО осуществляется разово для каждого пользователя компании. Повторная регистрация в ЭДО для исполнения последующих контрактов не требуется.
