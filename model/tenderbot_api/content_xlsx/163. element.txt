Перед тем как Заказчик увидит УПД необходимо пройти шаги: 
1) Поставщику Зарегистрироваться в ЭДО
2) Заказчику Зарегистрироваться в ЭДО
3) Поставщику сделать запрос на установку связи с Заказчиком по ЭДО
4) Заказчику принять приглашение к связи с Поставщиком по  контракту с помощью ЭДО
5) Поставщику отправить УПД
6) Поставщику подтвердить квиток отправки УПД
