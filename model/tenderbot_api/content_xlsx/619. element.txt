При блокировке компании Администраторами Портала соответствующая информация появится в виде уведомления в личном кабинете с указанием причины - основания блокировки. Администратор компании может заблокировать пользователя, при совершении данного действия у пользователя появится надпись "заблокирован" без указания срока блокировки, при этом компания не будет заблокирована.
