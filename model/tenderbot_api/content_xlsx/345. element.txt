Как заказчику посмотреть размещенные предложения всех поставщиков в закупке по потребности?
На странице закупки по потребности необходимо открыть раздел Предложения и просмотреть сделанные предложения, нажав кнопку Просмотреть.

Дополнительно для заказчика доступна возможность Выгрузить протокол на странице потребности. В протоколе будут отображены все предложения.
 

           Функционал просмотра протокола доступен по завершению приёма предложений в потребности.
Поставщик при размещении предложения указывает срок его действия, по истечению указанного срока предложение недоступно для заключения контракта и перестаёт отображаться для заказчика в разделе Предложения.
