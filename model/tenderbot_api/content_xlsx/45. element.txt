На Портале поставщиков пользователям доступен реестр поставщиков и реестр заказчиков.
 
Для поиска заказчиков пользователю необходимо в меню Портала поставщиков перейти в раздел «Заказчики», нажав на соответствующую кнопку.
 

 
Далее осуществить поиск по одному (или нескольким) из критериев (фильтров), указать которые можно в правой части раздела, и нажать кнопку «Найти».
