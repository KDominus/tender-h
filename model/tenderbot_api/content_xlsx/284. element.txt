Поиск предложений по СТЕ производится внутри карточки СТЕ.
 

 
Необходимо нажать на значок фильтра (1), на «Показать расширенные фильтры» (2), заполнить поля и нажать «Применить» (3).
 
Поиск предложений осуществляется по одному (или нескольким) из следующих критериев:
 




№


Наименование поля


Описание






1.                


Поставщик


необходимо ввести наименование поставщика или часть наименования (без «ООО», «ИП» и подобного как в виде сокращения, так и виде расшифровки)




2.                


Регион поставки


Выбор значения или значений осуществляется из справочника регионов




3.                


Стоимость единицы, ₽


Необходимо ввести значения «от» и «до» стоимости интересующих оферт




4.                


Период действия


Необходимо ввести значения «с» и «по» для периода действия интересующих предложений




5.                


НДС


Выбор значения или значений осуществляется из справочника;




6.                


Артикул


Необходимо ввести артикул или его часть в соответствующее поле;




7.                


СМП


при необходимости проставить отметку для поиска поставщиков, имеющих организационный признак «Субъект малого предпринимательства»




8.                


УИС


при необходимости проставить отметку для поиска поставщиков, имеющих организационный признак «Уголовно-исполнительная система»;




9.                


Организации инвалидов


при необходимости проставить отметку для поиска поставщиков, имеющих организационный признак «Организации инвалидов».




10.             


Для физических лиц


при необходимости проставить отметку для поиска предложений, доступных для физических лиц




 
При нажатии на кнопку "Сбросить"   поля формы очищаются от введенных критериев.
При установке одного из статусов, на странице отобразятся только те предложения, параметры которых соответствуют выбранным.
