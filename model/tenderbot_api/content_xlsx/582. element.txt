В настоящее время нет единой формы МЧД. 
Каждый сервис, использующий электронную подпись для подписания документов, будет оформлять МЧД по своей форме или определять порядок использования иных МЧД на своих сервисах.  
В общем виде процесс выглядит так: Руководитель организации заполняет форму доверенности. В ней указываются данные компании, ФИО сотрудника, его полномочия. Выставляется срок действия доверенности.  Руководитель подписывает МЧД своим сертификатом руководителя и получает соответствующий xml-файл доверенности для сотрудника.  Например: для работе в ЕИС нужно оформлять МЧД по форме, установленной ЕИС (с 01.10.2023 функционал формирования МЧД будет работать в ЕИС в полном объеме).
