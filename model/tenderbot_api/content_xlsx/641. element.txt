Рекомендуем проверить установлены ли необходимые полномочия в личном кабинете «Заказчик», «Специалист с правом подписи контрактов». Необходимо в столбце "действия" на странице потребности нажать на кнопку "Контракт", после чего будет возможно создание контракта.
Кроме того пользователь "Уполномоченного органа" с ролью "Организатор закупки" может создавать закупку от имени заказчика. В этом случае в закупке по потребности для такого заказчика не будет доступна возможность формирования контракта. Сформировать контракт для такой закупки сможет только "Организатор закупки".
 
В случае отображения только кнопки "Посмотреть" необходимо обратиться в службу технической поддержки Портала Поставщиков.
