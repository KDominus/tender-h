При прикреплении сертификата электронной подписи (далее - ЭП) в профиле пользователя может возникнуть ошибка Проверка правильности данных. Этот сертификат уже используется другим пользователем. Пожалуйста, выберите другой (Рисунок 1)
 

Рисунок 1
 
Данное сообщение означает, что этот сертификат ЭП уже привязан к другому пользователю, зарегистрированному на Портале поставщиков.
Рекомендовано авторизоваться на Портале при помощи ЭП.  Для этого необходимо на этапе авторизации на Портале поставщиков выбрать вкладку ЭП (Рисунок 2)
