Чтобы перейти в профиль пользователя, необходимо в выпадающем пользовательском меню нажать "Управление профилем" → "Профиль пользователя":
 

 
Страница "Профиль пользователя" содержит основные данные о пользователе:
 



Наименование поля 
Описание поля


ФИО пользователя 
В данном поле указывается фамилия, имя и отчество сотрудника компании, которому принадлежит данный профиль


Перечень полномочий
Полномочия - это разрешенные пользователю действия в системе. Полномочия для пользователя устанавливает администратор компании. 


E-mail (логин)

В данном поле должна быть указана электронная почта, на которую будут поступать уведомления от Портала поставщиков. 
Эта же электронная почта является логином для входа на Портал поставщиков.



Должность
Указывается должность сотрудника компании, которому принадлежит данный профиль


Подразделение
Указывается подразделение, в котором работает пользователь 


Состояние
Профиль пользователя может быть как активным, так и заблокированным. Заблокировать профиль пользователя может администратор компании.


Дата и время регистрации
Указывается дата и время создания/обновления профиля пользователя. Профиль может обновить, как сам сотрудник, которому принадлежит профиль, так и администратор компании.


Дата и время обновления профиля


Изображения профиля
В разделе "Прикрепленные документы" есть возможность добавить изображение профиля, которое отобразится в шапке Портала поставщиков, а так же будет отображаться как идентификация для других пользователей. 



Изображение профиля 
Для того, чтобы добавить изображение пользователя необходимо выполнить следующие шаги:
 
1. В блоке "Прикрепленные документы" нажать на кнопку "Добавить файл"
 
 
2. В открывшемся окне нажать кнопку "Выберите файл"
 

3. Найти и выбрать необходимое изображение формата jpg, jpeg или png на ПК,
 
4. Заполнить наименование файла, 
 
5. Нажать кнопку "Подтвердить" 
 

 
Чтобы загрузить фото профиля из раздела "Мои документы, необходимо: 
 
1.В разделе "Прикрепленные документы" нажать кнопку "Добавить файл" из раздела "Мои документы"
 

2. В открывшемся окне выбрать строку с необходимым фалов, нажав на неё,
 
3. Нажать кнопку "Выбрать".
 

Электронная подпись
Важно! ЭП должна принадлежать человеку, которому принадлежит данный (ФИО пользователя Портала поставщиков должна совпадать с ФИО держателя ЭП). Если ЭП принадлежит другому человеку, то необходимо создать нового пользователя организации. Для этого следует обратиться к пользователю организации, имеющему права администратора компании на Портале поставщиков.
 
Прикрепление электронной подписи
Для того, чтобы прикрепить электронную подпись, необходимо:
 

Нажать кнопку "Прикрепить ЭП" в правой части страницы профиля пользователя.


2. Подтвердить прикрепление электронной подписи, нажав кнопку "Да" 
 
 

3. Выбрать сертификат ЭП
 
4. Ввести текущий пароль пользователя
 
5. Нажать кнопку "Сохранить"
 

Смена электронной подписи 
Для того, чтобы сменить электронную подпись, необходимо:
 

В профиле пользователя нажать кнопку "Сменить ЭП"


2. В открывшемся окне подтвердить смену электронной подписи 
 

3. В открывшемся окне выбрать сертификат 
 
4. Ввести текущий пароль пользователя
 
5. Нажать кнопку "Сохранить"

Удаление электронной подписи
Важно! После удаления электронной подписи вы будете ограничены в правах на Портале и не сможете совершать юридически значимые действия.
 
Для того, чтобы удалить электронную подпись пользователя, необходимо в профиле пользователя: 
 

Нажать кнопку "Удалить ЭП"


 
 
2. В открывшемся окне подтвердить удаление ЭП, нажав кнопку "Да" 
 

Смена пароля
Для того, чтобы сменить пароль пользователя необходимо:
 
1. В профиле пользователя нажать кнопку "Смена пароля"

 
2. В открывшемся окне ввести Текущий пароль, новый пароль (дважды).
3. Нажать кнопку "Сменить пароль".
 

Смена e-mail
Важно! E-mail используется в качестве логина, при смене электронной почты логин будет изменен. E-mail должен быть уникальным на Портале поставщиков.
 
Для того, чтобы сменить электронную почту, необходимо: 
 
1. Нажать кнопку "Сменить E-mail"

 
2. В открывшемся окне указать текущий пароль и новую электронную почту
 
3. Нажать кнопку "Сохранить"
 

 
4. После этого на новую почту будет отправлено письмо о смене электронной почты с ссылкой. Для подтверждения смены электронной почты необходимо перейти по данной ссылке.
