Удалить закупку по потребностям в статусе «Ввод сведений» можно двумя способами: нажав на кнопку "Удалить"  из реестра закупок по потребностям, нажав на кнопку "Удалить"  на странице закупки по потребностям.
 
 
В модальном окне удаления потребности нажать на кнопку «Да» для подтверждения действия.
 
 
Для того, чтобы отменить уже опубликованную закупку необходимо перейти в реестр закупок по потребностям (Магазин → Мои закупки по потребностям) и нажать кнопку «Отменить» для потребности в статусе «Прием предложений».
После чего откроется модальное окно подтверждения отмены закупки по потребности.
 
 
После отмены закупки по потребности, ее статус изменится с «Прием предложений» на «Отменена».
