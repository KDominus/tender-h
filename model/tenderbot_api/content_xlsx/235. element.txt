Виды закупок, отображаемых в разделе "Единый реестр закупок"
В Едином реестре закупок отображаются закупки следующих типов:

Котировочные сессии;
Закупки по потребностям;
Конкурентные процедуры;
В2В закупки.


 
Карточка закупки 
Карточка закупки содержит следующие данные:
 



Наименование поля карточки
Описание поля карточки


Тип закупки
В данном поле указывается тип закупки


Номер закупки
Указывается номер закупки


Наименование закупки
Наименование или предмет закупки


Наименование заказчика
Краткое наименование и ИНН заказчика 


Даты проведения
Период проведения закупки


Начальная цена 
Цена, которую заказчик установил до начала проведения закупки


Регион 
Регион заказчика, который осуществляет данную закупку


Закон
Закон, регламентирующий проведение данной закупки



Карточка котировочной сессии

 
Карточка котировочной сессии содержит следующие дополнительные данные:
 



Наименование поля карточки
Описание поля карточки


Статус закупки

Текущий статус котировочной сессии. Котировочная сессия может иметь один из следующих статусов: 

Активная (поставщик может принять участие в данной КС),
Проведена (подача заявок завершилась, принять участие в данной котировочной сессии уже невозможно),
Снята с публикации (котировочная сессия была снята с публикации заказчиком или администратором Портала поставщиков),
Не состоялась (котировочная сессия признана не состоявшейся (подробнее см. Регламент ведение Портала Поставщиков) 




Количество ставок
Количество ставок в котировочной сессии, которые сделали поставщики за весь период проведения


Возможная ставка
Текущая возможная ставка, которую может сделать поставщик 



Карточка закупки по потребности и B2B закупки

 
Карточка закупки по потребности и В2В закупки содержит следующие дополнительные данные:
 



Наименование поля карточки
Описание поля карточки


Статус закупки

Текущий статус закупки по потребности. Закупка по потребности может иметь один из следующих статусов: 

Активная (поставщик может принять участие в данной закупке по потребности),
Проведена (подача заявок завершилась, принять участие в данной закупки по потребности уже невозможно)




Количество предложений

Количество предложений в закупке по потребностям, которые направили поставщики за весь период проведения




Карточка закупок по конкурентным процедурам

 
Карточка закупки по конкурентным процедурам содержит следующие дополнительные данные:
 



Наименование поля карточки
Описание поля карточки


Статус закупки

Текущий статус закупки по 223-ФЗ и 44-ФЗ.
