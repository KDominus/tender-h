Перед проверкой документа происходит автоматическое сопоставление тестовых данных, хранящихся в используемом типе документе, и установленных управляющих элементов.
При проверке шаблона пользователь может проконтролировать автоматическое «раскрытие» установленных ранее "служебных меток" в шаблоне, перед его публикацией. 
Для начала проверки необходимо нажать кнопку "Проверить»" и указать путь для сохранения черновика шаблона.
 

 
Скаченный черновик шаблона в формате "*docx" необходимо открыть в текстовом редакторе MS Word.
По итогу файл должен открыться корректно с обычными управляющими элементами и повторяющимися элементами
