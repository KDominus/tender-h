При регистрации на Портале поставщиков, пользователь может самостоятельно выбрать вариант регистрации:

С использованием электронной подписи;
Упрощенный вариант регистрации. Данный вариант регистрации позволит пользователю просматривать на Портале поставщиков общую информацию, в том числе список активных котировочных сессий, а также настроить подписку приглашений на котировочные сессии. Для получения расширенных функций при использовании Портала поставщиков, например, создание оферт и участие в котировочных сессиях, пользователю необходимо в личном кабинете прикрепить электронную подпись.

Для регистрации на Портале по электронной подписи:

Нажмите "Регистрация";
Заполните поле логин, выберете сертификат электронной подписи, укажите действующий адрес электронной почты (если в выбранной электронной подписи указан адрес электронной почты, он автоматически будет внесен в соответствующее поле на форме регистрации);
Нажмите на кнопку "Зарегистрироваться". На указанный адрес электронной почты будет отправлен временный пароль, с помощью которого возможно завершить регистрацию на Портале. Для доступа к функциям Портала необходимо перейти в свой профиль, далее в раздел "Заявка на регистрацию". Затем заполнить необходимые поля, отправить заявку на регистрацию и дождаться её согласования. После обработки заявки в Личный кабинет поставщика поступит уведомление.
