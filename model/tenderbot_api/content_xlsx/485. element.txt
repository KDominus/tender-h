На Портале поставщиков возможен упрощенный вариант регистрации для поставщиков - без использования сертификата электронной подписи (далее ЭП). Данный вариант регистрации позволит пользователю просматривать на Портале поставщиков общую информацию, в том числе список активных котировочных сессий (далее КС), а также, настроить подписку приглашений на КС. Для получения расширенных функций при использовании Портала поставщиков, например, создание предложений (оферт) и участие в КС, пользователю необходимо в личном кабинете прикрепить ЭП.
Упрощённая регистрация для заказчика на Портале поставщиков не предусмотрена.
Сертификат ключа ЭП должен быть изготовлен аккредитованными удостоверяющими центрами, перечень, которых представлен на портале уполномоченного федерального органа в области использования ЭП.
