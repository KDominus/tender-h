Формирование и редактирование шаблонов типовых документов осуществляется пользователем уполномоченного органа в разделе по работе с типовыми документами.
Пользователю необходимо перейти в блок «Управление 2.0», в раздел «Типовые документы». 
 

 
Для начала работы с шаблонами пользователю необходимо добавить в систему новый шаблон, на основании которого в дальнейшем можно будет формировать типовые документы.  
 

Для создания нового шаблона необходимо нажать на кнопку «Новый шаблон»  . Далее пользователю необходимо прописать название шаблона и выбрать тип документа
 

После заполнения необходимых данных пользователь должен нажать на кнопку «Создать» 
 

В результате действия добавленный шаблон отображается в реестре списка шаблонов в статусе «Проект»
 

Кнопка «Отмена» (при создании шаблона) очищает поля от пользовательских данных и скрывает меню создания шаблона.
Любой шаблон отображается в системе в виде карточки и всегда отображается выше других шаблонов, добавленных раньше него. Вверху карточки отображается данное на предыдущем шаге имя шаблона и его статус (1). Для шаблона в системе доступно три статуса. Причем, каждый статус имеет свой собственный цвет, отображающийся в виде прямоугольника в левом верхнем углу карточки:

Проект (темно синий) – подготовка шаблона к публикации;
Активен (зеленый) – шаблон опубликован;
В архиве (серый) – шаблон отправлен в архив, все работы с ним завершены. 


Ниже названия располагается информация о типе документа и его версии (2). Внизу находятся 5 кнопок для работы с черновиком шаблона (3): «Загрузить документ», «Скачать черновик шаблона», «Загрузить шаблон», «Проверить», «Опубликовать». В отдельном блоке справа отображаются кнопки управления шаблоном (4): «Редактировать», «Удалить»
