Данная ошибка возникает в том случае, если пользователь с таким логином уже зарегистрирован.
 
По умолчанию логином является адрес электронной почты, на который регистрируется профиль. Вы можете воспользоваться функцией «Забыли пароль?» для восстановления доступа.
Регистрацию региональных заказчиков на Портале поставщиков производят уполномоченные органы (УО) региона. Для восстановления логина Вы можете обратиться в УО по Вашему региону.
Также Вы можете обратиться в Службу технической поддержки Портала поставщиков с запросом на восстановление доступа.
