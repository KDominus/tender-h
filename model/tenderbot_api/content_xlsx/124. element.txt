При получении данного сообщения, необходимо проверить корректность установки программного обеспечения для считывания электронной подписи (криптопровайдер).
 
В случае необходимости, требуется его настроить.
 
По вопросам настройки и переустановки криптопровайдера пользователь может обратиться к системному администратору своей организации или к поставщику программного обеспечения для считывания электронной подписи (криптопровайдера).
