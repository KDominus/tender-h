Для просмотра детальной информации о поставщике требуется зайти в раздел "Организации" - "Поставщики" главного меню Портала. Далее нажать на наименование организации поставщика, которое является гиперссылкой для перехода в карточку поставщика. 
 
Поиск поставщиков в реестре
Поиск поставщиков осуществляется по одному (или нескольким) из следующих критериев (фильтров), указать которые можно в правой части раздела. Чтобы осуществить поиск, необходимо: 
 
1. Ввести критерии поиска в область поиска, 
 
2. Нажать кнопку "Найти".
 
Доступные критерии поиска:



Наименование поля поиска
Описание поля поиска


Вид продукции
Поиск будет осуществляться по поставщикам, у которых есть оферты с указанными в поле видами продукции . В данном поле используется справочник продукции Портала поставщиков. 


Категория продукции
Поиск будет осуществляться по поставщикам, у которых есть оферты с указанными в поле категориями продукции . В данном поле используется справочник КПГЗ.


Регион регистрации
Поиск будет осуществляться по поставщикам, которые зарегистрированы в выбранных в поле регионах


Регион поставки
Поиск будет осуществляться по поставщикам, у которых есть оферты с выбранным в поле регионом поставки


Наименование
Поиск будет осуществляться по поставщикам, у которых в наименовании есть указанный в поле набор символов


ИНН
Поиск будет осуществляться по поставщикам, у которых в ИНН имеется указанный в поле набор символов


ОГРН
Поиск будет осуществляться по поставщикам, у которых в ОГРН имеется указанный в поле набор символов


КПП
Поиск будет осуществляться по поставщикам, у которых в КПП имеется указанный в поле набор символов


Тип поставщика

Поиск будет осуществляться по поставщикам с типом, указанном в поле. Доступные для поиска типы: 

Юридическое лицо, 
Физическое лицо,
Индивидуальный предприниматель.




Дата регистрации
Поиск будет осуществляться по поставщикам, у которых дата регистрации входит в указанный в поле временной период


Организационные признаки 

Результаты поиска будут включать в себя поставщиков, обладающих хотя бы одним из выбранных признаков: 

СМП (Субъекты малого и среднего предпринимательства),
УИС (Предприятие, относящееся к уголовно-исполнительной системе),
Организация инвалидов, 
Социально-ориентированные организации.




Статусы поставщика

Для поиска доступны следующие статусы: 

Лидер Портала поставщиков,
Официальный дилер,
Поставщик инновационной продукции.




Год регистрации 
Поиск будет осуществлен по выбранному году регистрации компании на Портале поставщиков


Зарегистрирован на Портале поставщиков
Поиск будет осуществлен по признаку, показывающему, что поставщик зарегистрирован на Портале, а не во внешней системе


Производитель продукции
Поиск будет осуществлен по признаку, показывающему, что поставщик обладает статусом "Производитель продукции" на Портале


Цена в офертах завышена
Поиск будет осуществлен по признаку, показывающему, что у поставщика есть оферты, в которых цены завышены


Не завершивший регистрацию 
Поиск будет осуществлен по признаку, показывающему, что поставщик зарегистрировался на Портале поставщиков, но не отправил заявку на регистрацию компании/не прикрепил электронную подпись



Страница поставщика
Для просмотра детальной информации о поставщике требуется нажать на наименование организации поставщика.
После чего откроется публичная страница организации с более детальной информацией, а так же статистикой по контрактам.
