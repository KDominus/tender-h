Подсистема разработана в клиент-серверной архитектуре с использованием «тонкого клиента», таким образом, использование дистрибутивного носителя данных специального программного обеспечения на АРМ пользователя не требуется. 
 
Для начала работы с Порталом необходимо и достаточно наличие установленного ОПО (Таблица 2). Описание установки КриптоПро CSP 3.9 приведено в п.3.2.2, установка остальных компонентов осуществляется в соответствии с инструкцией производителя.
