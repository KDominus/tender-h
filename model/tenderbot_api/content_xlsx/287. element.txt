Свои заявки на прямую закупку Заказчик может отслеживать в разделе «Магазин» → «Мои заявки» с помощью всплывающего пользовательского меню:
 

 
Если пользователь выбирает заявку со статусом «Черновик», то ему становятся доступны кнопки «Направить поставщику» и «Удалить».
 

 
Если пользователь выбирает заявки с другими статусами, то становится доступна кнопка «Создать новую на основании заявки», при нажатии на которую открывается страница создания заявки с уже заполненными полями.
