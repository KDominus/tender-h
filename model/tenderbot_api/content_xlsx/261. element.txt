Для внесения изменений в опубликованный шаблон потребуется создать для него новую версию. Данный шаг не обязует пользователя актуализировать именно первоначальный файл шаблона. Сервис допускает как редактирование предыдущего шаблона, так и загрузку абсолютно нового файла для следующей версии. Однако, изменить тип документ будет нельзя.
Для добавления новой версии шаблона необходимо нажать кнопку «Новая версия» и подтвердить действие:
 

 
В результате будет добавлена новая версия выбранного шаблона (1) в статусе «Проект» (2):
 
 
 
Для публикации шаблона в новой версии необходимо проделать те же шаги, что и для подготовки нового шаблона (Подготовка шаблона документа).
