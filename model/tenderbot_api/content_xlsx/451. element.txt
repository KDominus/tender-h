Номер закупки по конкурентной процедуре для заполнения поля «Закупка» в заявке на банковскую гарантию можно узнать в Едином реестре закупок. Для этого необходимо:
1. В меню Портала поставщиков выбрать раздел «Единый реестр закупок» подраздел «Конкурентные процедуры» (Рисунок 1)
 

Рисунок 1
2. В реестре найти процедуру, по которой требуется оформить банковскую гарантию и скопировать номер закупки (Рисунок 2)
 

Рисунок 2
Подробное описание Единого реестра закупок приведено в руководствах пользователя «Инструкция для поставщика» и «Инструкция для заказчика».
Номер контракта по конкурентной процедуре для заполнения поля «Контракт» в заявке на банковскую гарантию можно узнать в Реестре контрактов. Для этого необходимо:
1. В меню Портала поставщиков выбрать раздел «Контракты» подраздел «По конкурентным процедурам» (Рисунок 3)
 

Рисунок 3
2. В реестре найти контракт, по которому требуется оформить банковскую гарантию и скопировать номер контракта (Рисунок 4).
Подробное описание Реестра контрактов приведено в руководствах пользователя «Инструкция для поставщика» и «Инструкция для заказчика».
