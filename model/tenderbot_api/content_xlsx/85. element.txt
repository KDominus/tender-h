Для направления заявки на добавление стандартной товарной единицы (далее - СТЕ) необходимо указать не менее 4-х (четырех) характеристик. Характеристики должны конкретно описывать предлагаемый товар/услугу/работу. Для добавления характеристики СТЕ необходимо нажать на кнопку «Изменить характеристики» в карточке создания СТЕ.
