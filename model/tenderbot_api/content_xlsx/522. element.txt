При заполнения юридического и фактического адреса в профиле компании на Портале поставщиков в поле Адрес необходимо ввести значение вручную. Далее, выбрать адрес из выпадающего списка (Рисунок 1)

Рисунок 1
Для городов федерального значения рекомендуемый порядок ввода адреса: улица, дом.
Для других регионов рекомендуемый порядок ввода адреса: город, улица, дом.
В случае отсутствия адреса в выпадающем списке рекомендовано:
●      Снять чекбокс с поля Использовать подсказки (Рисунок 2)

Рисунок 2
●     Заполнить вручную значения разделов Юридический/Фактический адрес Город/н.п., Улица, Дом, Квартира/Офис (Рисунок 3)

Рисунок 3
