На странице закупки по потребности необходимо открыть раздел Предложения и просмотреть сделанные предложения, нажав кнопку Просмотреть (Рисунок 1)
 
Рисунок 1
Дополнительно для заказчика доступна возможность Выгрузить протокол на странице потребности. В протоколе будут отображены все предложения (Рисунок 2)
 
Рисунок 2
Функционал просмотра протокола доступен по завершению приема предложений в потребности.
Поставщик при размещении предложения указывает срок его действия, по истечению указанного срока предложение недоступно для заключения контракта и перестаёт отображаться для заказчика в разделе Предложения.
