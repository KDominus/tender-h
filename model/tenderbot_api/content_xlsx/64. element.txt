Для загрузки прайс-листа на Портал поставщиков необходимо войти в личный кабинет пользователя, и нажать на имя пользователя. Далее, в выпадающем списке, выбрать "Оферты" - "Импорт YML прайс-листов". Затем необходимо загрузить прайс-лист с помощью кнопки "Загрузить".
