Ознакомиться с разделом B2B закупки можно через Меню на Портале поставщиков, выбрав раздел Единый реестр закупок и подраздел B2B-закупки (Рисунок 1)
 
 

Рисунок 1
 
При выборе раздела B2B закупки, отображаются все закупки по потребностям на Портале поставщиков, где заказчиками выступают коммерческие предприятия (Рисунок 2)
 
 

Рисунок 2
 
В данном разделе организовано взаимодействие с такими площадками, как:
● "ВТБ бизнес-коннект" (маркетплейс для малого и среднего бизнеса)
● SberB2B (сервис электронной торговой площадки "Сбербанк-АСТ")
● "Единая электронная торговая площадка" (Росэлторг, АО "ЕЭТП")
Для участия в подобной закупке может потребоваться переход во внешнюю систему.
