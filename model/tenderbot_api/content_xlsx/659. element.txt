Для проведения закупок малого объема заказчику необходимо выбрать оферту через:

электронный магазин системы ЕАИСТ (для московских заказчиков);
раздел «Товары» (для региональных заказчиков). В данном разделе отображены СТЕ (стандартные торговые единицы) продукции/услуги, которые содержат предложения (оферты) поставщиков.

Оферты поставщиков отображаются, если им присвоен статус «Зарегистрирована». Рекомендуем выбирать оферту, срок действия по которой оканчивается после завершения закупочной процедуры.
 
  
 
  
 
Не отображаются в перечне доступных оферт для выбора под закупочные процедуры малого объема:

оферты, созданные по результатам котировочной сессии;
оферты со статусом «Ввод сведений» и «В архиве».
