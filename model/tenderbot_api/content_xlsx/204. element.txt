Для удаления Поставщиком поданного предложения необходимо:
1. Перейти на страницу закупки по потребностям,
2. Перейти на вкладку «Предложения»,
3. Нажать на кнопку «Удалить».

4. В окне подтверждения удаления, нажать кнопку "Да": 

Важно! Удаление предложения, поданного Поставщиком возможно только для потребности в статусе «Прием предложений». В статусе «Прием предложений завершен» удалить предложения нельзя.
