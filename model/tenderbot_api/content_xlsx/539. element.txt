На форме «Выбор ОКОПФ» для ЮЛ необходимо выбрать значение из справочника ОКОПФ (Рисунок 1) и нажать на кнопку «Продолжить» (Рисунок 2).

Далее, откроется форма «Выбор ролей» где пользователь может выбрать необходимые роли нажав на кнопку «Выбрать роль» (Рисунок 3) и перейти к следующему шагу:



Регистрация роли «Поставщик»
Регистрация роли «Заказчик»;
Регистрация роли «Производитель».



 

Рисунок 3
 
  Если процесс регистрации профиля был прерван, пользователь может возобновить его. Для этого необходимо перейти в меню личного кабинета в раздел «Управление профилем» и нажать на кнопку «Заявка на регистрацию профиля» (Рисунок 4).
