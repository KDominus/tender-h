Для того, чтобы создать документ в разделе "Мои документы" необходимо:
 
1. Нажать кнопку "Создать" в шапке раздела: 
 

 
 
2. Затем откроется форма создания нового документа: 
 

 
 
3. В открывшейся странице заполнить следующие поля и загрузить файл в соответствии с выбранным типом документа:
 



Наименование поля для заполнения
Описание поля для заполнения


Наименование документа
Указывается наименование загружаемого документа


Номер документа
Указывается номер документы. Это может быть регистрационный номер, присвоенный отделом документационного обеспечения организации, номером может также порядковый номер документа на Портале поставщиков


Тип документа

На Портале доступны следующие виды документов: 

Изображения, 
Лицензия,
Организационный документ,
Прайс-лист поставщика,
Проект контракта, 
Сертификат,
Прочее.




Примечание

В данном поле можно указать комментарий к загружаемому документу 



Файл

В зависимости от выбранного в поле "Тип документа" значения, пользователю будут доступны определенные форматы загружаемого файла: 

Изображения - форматы *.jpg, *.jpeg, *.png, *.gif, *.yml, 
Лицензия - форматы *.jpg, *.txt,*.doc, *.rtf, *.docx, *.xls, *.xlsx, *.gif, *.pdf, *.png, *.tiff, *.pptx, *.ppt , *.yml ,
Организационный документ - форматы *.jpg, *.txt,*.doc, *.rtf, *.docx, *.xls, *.xlsx, *.gif, *.pdf, *.png, *.tiff, *.pptx, *.ppt ,*.yml ,
Прайс-лист поставщика - формат *.yml,
Проект контракта - форматы *.jpg, *.txt,*.doc, *.rtf, *.docx, *.xls, *.xlsx, *.gif, *.pdf, *.png, *.tiff, *.pptx, *.ppt ,*.yml ,
Сертификат - форматы *.jpg, *.txt,*.doc, *.rtf, *.docx, *.xls, *.xlsx, *.gif, *.pdf, *.png, *.tiff, *.pptx, *.ppt ,*.yml ,
Прочее - формат





 
4. После того, как все поля будут заполнены, необходимо нажать кнопку "Сохранить"^
