Прикрепление контракта к оферте возможно только в том случае, если она находится в статусе «Ввод сведений».
Реализована возможность прикрепления файла проекта контракта к одной или нескольким офертам. Для этого необходимо перейти в раздел «Мои оферты» и выбрать вариант отображения «Список».
 

 
Далее установить флажки в строках оферт, к которым необходимо прикрепить загруженный проект контракта и внизу страницы нажать на одну из ссылок ("Добавить файл из раздела "Мои документы"" или "Добавить файл"), затем выбрать и загрузить файл контракта.
Далее требуется нажать на кнопку «Загрузить проект контракта».
 
 
В результате файл с проектом контракта прикрепляется к выделенным офертам и на экране отобразится сообщение об успешной операции.
