Если опубликованный шаблон перестал быть актуальным, то его можно отправить в архив. 
Важно! Вернуть из архива шаблон будет нельзя. По шаблону в статусе "В архиве" нельзя создавать документы.
Чтобы отправить опубликованный шаблон в архив необходимо нажать на одноименную кнопку и подтвердить действие на странице.
 

 
В результате действия шаблон перейдет в статус "В архиве" (1). Внизу будет добавлена дата события (2).
