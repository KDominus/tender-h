Раздел "Мои документы" или электронная библиотека предназначен для систематизированного хранения неограниченного количества однотипных файлов/документов организации с целью минимизации и исключения дублирующих действий по учету документов. Раздел также снабжен средствами навигации и поиска загруженных пользователями документов/файлов.
 
Документы и файлы в разделе делятся по типам, могут нумероваться, а также редактироваться всеми пользователями организации. В связи с этим в разделе есть возможность просматривать историю изменения файла (таблицу с датой редактирования сведений, а также ФИО пользователя, совершившего изменения). 
 
Переход в раздел "Мои документы"
Перейти в раздел можно с помощью пользовательского всплывающего меню, нажав кнопку "Документы" → "Управление документами": 
 

 
Раздел "Мои документы" содержит все документы, загруженные пользователями организации.
 

 
Таблица результатов поиска содержит следующие сведения о документе:
 



Наименование поля
Описание поля


Отметка выбора документа
С помощью данной отметки можно совершать массовые операции над документами. Например, установив отметку в шапке портала (выбрав все документы на странице), можно удалить все документы на странице раздела 


Наименование документа 
Пользовательское наименование документа (не прикрепленного файла) 


Дата документа
Дата, когда документ был загружен 


Тип документа 

Тип документа, который выбрал пользователь на этапе создания. На текущий момент система поддерживает следующие типы документов: 

Изображения - форматы *.jpg, *.jpeg, *.png, *.gif, *.yml, 
Лицензия - форматы *.jpg, *.txt,*.doc, *.rtf, *.docx, *.xls, *.xlsx, *.gif, *.pdf, *.png, *.tiff, *.pptx, *.ppt , *.yml ,
Организационный документ - форматы *.jpg, *.txt,*.doc, *.rtf, *.docx, *.xls, *.xlsx, *.gif, *.pdf, *.png, *.tiff, *.pptx, *.ppt ,*.yml ,
Прайс-лист поставщика - формат *.yml,
Проект контракта - форматы *.jpg, *.txt,*.doc, *.rtf, *.docx, *.xls, *.xlsx, *.gif, *.pdf, *.png, *.tiff, *.pptx, *.ppt ,*.yml ,
Сертификат - форматы *.jpg, *.txt,*.doc, *.rtf, *.docx, *.xls, *.xlsx, *.gif, *.pdf, *.png, *.tiff, *.pptx, *.ppt ,*.yml ,
Прочее - форматы *.jpg, *.txt,*.doc, *.rtf, *.docx, *.xls, *.xlsx, *.gif, *.pdf, *.png, *.tiff, *.pptx, *.ppt ,*.yml .




№ версии 
Номер загруженной версии документа


История 
При нажатии на кнопку "Посмотреть историю" открывается окно с историей изменений данного документа. В окне отображается таблица с датой изменений, наименованием операции и пользователем организации, сделавшим изменения



 
Если в столбце "По умолчанию" установлен флажок, то данный файл будет автоматически прикрепляться к создаваемым на Портале поставщиков документам.
